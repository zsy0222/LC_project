# 垃圾分类处理工艺知识库

> 基准情景：卫生填埋（0 kg CO₂e/吨）。净减排量为负数表示相对于填埋减少的碳排放量。
> 厨余数据来源：DB4403/T 468—2024《厨余垃圾处理项目碳排放核算指南》及清华大学深圳国际研究生院与深圳市城管宣教和发展研究中心联合研究报告（2025）。涵盖范围一、二、三全生命周期排放及碳补偿。
> 代码入口：`src/config.py` → `TREATMENT_KNOWLEDGE`

## 一、厨余垃圾处理工艺

### 基准情景：卫生填埋

| 属性 | 内容 |
|------|------|
| 碳减排量 | **0 kg CO₂e/吨**（基准情景） |
| 原理 | 厨余垃圾在填埋场厌氧分解产生大量甲烷，每吨填埋约产生 310~350kg CH₄，折合 8680~9800kg CO₂ 当量。即使考虑填埋气收集利用，仍有大量甲烷泄漏，是碳排放最高的处理方式。 |
| 数据来源 | DB4403/T 468—2024、《省级温室气体清单编制指南》（2011） |

---

### 1.1 厌氧消化工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-90.82 kg CO₂e/吨**（范围：-65~-120kg） |
| 原理 | 避免填埋产生的甲烷排放（甲烷温室效应是CO₂的28倍），同时沼气发电/提纯生物天然气替代化石能源，沼渣沼液生产有机肥替代化肥。其中甲烷控排贡献约60%，能源替代贡献约30%，肥料替代贡献约10%。 |
| 技术优势 | 减排效果最优，产物附加值高（沼气发电+有机肥双产出）；技术成熟，国内外大规模应用；可与市政燃气管网配套。沼气收集率每提高10%可增加减排约15kg/t。 |
| 数据来源 | 《典型厨余垃圾资源化处理项目碳排放分析》（《环境卫生工程》2025年第3期）、DB4403/T 468—2024 |

---

### 1.2 发酵产酸工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-81.04 kg CO₂e/吨**（范围：-70~-95kg） |
| 原理 | 通过生物发酵将厨余垃圾中的有机质转化为挥发性脂肪酸（VFA），作为污水处理厂的生物碳源，替代传统葡萄糖、乙酸钠等化工碳源生产过程的高碳排放。 |
| 技术优势 | 处理周期短（仅2~3天，为厌氧消化的1/10），占地面积小；特别适合与大型污水处理厂协同建设；产物附加值高（工业碳源市场价格优于沼气）。 |
| 数据来源 | 同上 |

---

### 1.3 三相分离协同焚烧工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-80.96 kg CO₂e/吨**（范围：-70~-90kg） |
| 原理 | 先通过三相分离技术提取厨余垃圾中的油脂生产生物柴油，分离出的有机固渣送入垃圾焚烧厂焚烧发电，减少填埋和化石能源使用。其中油脂制生物柴油贡献约55%的减排量。 |
| 技术优势 | 利用已有焚烧设施降低独立建厂成本；油脂回收+焚烧发电双路径产出；适合与垃圾焚烧厂邻近建设。 |
| 数据来源 | 同上 |

---
### 1.4 好氧堆肥工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-25~-50 kg CO₂e/吨** |
| 原理 | 通过好氧微生物分解将厨余垃圾转化为腐殖质有机肥，避免填埋甲烷排放，同时替代化肥生产。高温阶段可达50~65℃，能有效杀灭病原菌。 |
| 技术优势 | 技术简单成熟，运行成本低；减量率可达90%以上；现代一体化设备可压缩周期至5~10天；可分散式就地处理，避免运输成本。局限性：占地面积较大，需配套除臭系统。 |
| 数据来源 | 中科院生态环境研究中心《我国厨余垃圾处理模式的综合比较和优化策略》（2021）、中国城市环境卫生协会（2024） |

---

### 1.5 直接混合焚烧工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-3.01 kg CO₂e/吨**（范围：+10~-20kg，低效项目可能出现增排） |
| 原理 | 焚烧产生的热量用于发电，替代部分化石能源。但厨余垃圾含水率高（60%~80%），焚烧前需要大量能耗进行干燥，且未充分利用其资源化潜力，整体减排效果极弱。 |
| 技术优势 | 处理简单快速，利用现有焚烧厂设施，不需独立建厂。局限性：热值低、含水率高导致焚烧效率差，资源化程度最低，不推荐作为长期策略。 |
| 数据来源 | 同上 |

---

### 1.6 黑水虻+厌氧消化集成工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-120~-150 kg CO₂e/吨** |
| 原理 | 先通过黑水虻转化厨余垃圾中的易降解有机质，剩余的难降解残渣再进行厌氧消化产沼气。与单一厌氧消化相比，可降低总能耗13%，碳足迹减少34%。 |
| 技术优势 | 减排量最高的组合工艺路线；产品多元化（沼气+昆虫蛋白+有机肥）；消化液产量减少14%，缓解后端处理压力；资源消耗减少73%。 |
| 数据来源 | 《生物通》"昆虫养殖耦合厌氧消化在餐厨垃圾生物精炼中的环境效益评估"（2025） |



### 1.7 蚯蚓堆肥工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-30~-50 kg CO₂e/吨** |
| 原理 | 蚯蚓（赤子爱胜蚓）吞食厨余有机质，在肠道微生物协同下分解为蚯蚓粪——高腐殖质、富氮磷钾的优质有机肥，同时产出虫体蛋白。常温运行（15~25℃），避免高温堆肥NH₃与N₂O排放，氮素保留率比好氧堆肥高30%~50%。 |
| 技术优势 | 设备极简——泡沫箱加蚯蚓即可启动，无异味可室内运行；蚯蚓粪市场价2~5元/kg；适合社团或班级管理，教学与环保双价值。局限：处理周期2~4周，对盐分与辣椒素敏感，需预发酵去刺激性。 |
| 数据来源 | 《现代农业科技》2023年第21期"蚯蚓资源化处理厨余垃圾的研究进展"（王煦、何欢、郭亮、郑爱芳）、中国农业大学有机循环研究院（2024） |

## 二、快递纸箱（废纸）处理工艺

### 基准情景：卫生填埋

| 属性 | 内容 |
|------|------|
| 碳减排量 | **0 kg CO₂e/吨**（基准情景） |
| 原理 | 废纸在填埋场厌氧分解产生甲烷（温室效应是CO₂的28倍），每吨填埋约产生778kg CO₂ 当量，是碳排放最高的废纸处理方式。 |
| 数据来源 | 《省级温室气体清单编制指南》（2011） |

---

### 2.1 回收制浆再生工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-1200~-1600 kg CO₂e/吨**（再生瓦楞纸约-1580，再生文化纸约-1410，再生白板纸约-950） |
| 原理 | 替代原生木浆生产，减少4~5m³木材消耗及制浆过程60%~70%的能源消耗和化学品使用，同时避免填埋产生的甲烷排放。 |
| 技术优势 | 减排效果极佳，是废纸最优先的处理路径；技术成熟，产业链完善，再生产品有稳定市场；每吨废纸再生可节约17棵大树。 |
| 数据来源 | 中国造纸协会《中国造纸工业碳排放核算指南》（2022）、清华大学环境学院《中国废纸回收利用碳减排效益评估研究》（2023） |

---

### 2.2 热解制生物炭工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-427 kg CO₂e/吨** |
| 原理 | 在缺氧环境下将废纸热解为稳定的生物炭，实现长达百年的碳封存；生物炭还可改良土壤，提升土壤碳汇能力。 |
| 技术优势 | 实现长期碳封存（百年尺度）；产物生物炭可改良土壤酸性、保水保肥；热解气可作为辅助能源。 |
| 数据来源 | 《环境工程技术学报》"生活垃圾低值可回收物分类及处理处置的碳减排潜力研究"（2024） |

---

### 2.3 焚烧发电工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-690 kg CO₂e/吨** |
| 原理 | 焚烧产生的热量用于发电，替代化石能源；废纸中的碳为生物成因，焚烧排放的CO₂属于碳循环的一部分，不计入化石碳排放。 |
| 技术优势 | 处理量大，减容效果显著（减容90%+）；利用现有焚烧设施，技术成熟。局限性：资源化程度低于制浆再生。 |
| 数据来源 | IPCC 国家温室气体清单指南（2019修订版）、中国环境科学研究院 |

---


### 2.4 蛋托/育苗钵模塑工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-700~-900 kg CO₂e/吨** |
| 原理 | 废纸箱碎解打浆后直接在模具中热压成型为蛋托、育苗杯等纸模产品，省去再生纸的漂白、抄纸工序。全生命周期GWP仅0.09 kg CO₂ eq/件，为PVC蛋托的12.5%，产品废弃后可自然降解或再次回收制浆。 |
| 技术优势 | 工艺极简，桌面级打浆机加热压模具即可在校内操作；每件减排约87%碳排放；产品为校园花圃刚需耗材，无需外部销售渠道。局限：强度低于注塑制品，遇水软化需配合防水涂层。 |
| 数据来源 | 大连工业大学《包装学报》"鸡蛋包装生产工艺的生命周期评价"（霍李江、赵昱，2021）、纸浆模塑行业LCA综合分析（pulpmoldingchina.cn） |

---

### 2.5 蘑菇培养料制备工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-500~-700 kg CO₂e/吨** |
| 原理 | 废纸撕碎后作为碳源主料，与麸皮、石膏混合制成菌包栽培食用菌。菌丝降解纤维素转化为子实体，菌渣为富含菌体蛋白的优质有机肥。烟台大学实验证实废纸培养基菌丝生长速率0.31 cm/d，生物学效率71.29%，与传统棉籽壳无显著差异。 |
| 技术优势 | 生长过程可视化，教学价值极高；菌包制作简单无需大型设备；产品可食用或义卖；菌渣二次堆肥或喂蚯蚓，形成无废闭环。局限：废纸油墨残留需关注食品安全，建议选用无印刷废纸箱。 |
| 数据来源 | 烟台大学《环境与发展》2024年第1期"废纸生物利用栽培多脂鳞伞初探"、中国食用菌协会 |


### 2.6 废纸纤维素隔热材工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-1300~-1500 kg CO₂e/吨** |
| 原理 | 废纸经碎浆、添加硼酸阻燃剂后干燥成型为纤维素隔热棉/板（密度30~60 kg/m³，导热系数0.035~0.040 W/m·K），替代玻璃棉和岩棉等高温冶炼隔热材料。WANG等（2023）通过全生命周期评价验证，纤维素隔热材在Passivhaus建筑中碳足迹远低于传统隔热材；法国OneTribe项目实测屋顶隔热减排92%、墙体减排88%，年减排约25,000 tCO₂e。 |
| 技术优势 | 减排幅度极大（88%~92% vs 矿物棉）；原料零成本（校园废纸/旧试卷）；防火等级B1可满足建筑规范；产品寿命30年以上，废弃后可再次回收。局限：需硼酸盐阻燃处理，潮湿环境需防潮层配合。 |
| 数据来源 | WANG & WANG, Sustainability (MDPI) 2023, "Can Paper Waste Be Utilised as an Insulation Material" (DOI:10.3390/su152215939)、OneTribe/Riverse法国纤维素隔热碳信用项目实测数据（2024） |

## 三、塑料处理工艺

### 基准情景：卫生填埋

| 属性 | 内容 |
|------|------|
| 碳减排量 | **0 kg CO₂e/吨**（基准情景） |
| 原理 | 普通塑料在填埋场难以降解，长期封存化石碳，但会产生严重的微塑料污染；部分易降解塑料会产生甲烷，每吨填埋约产生2.40t CO₂ 当量。 |
| 数据来源 | 《中国塑料》期刊（2025） |

---

### 3.1 物理回收再生（造粒）工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-1900~-2400 kg CO₂e/吨**（范围：-1200~-2800kg，取决于塑料种类和回收纯度） |
| 原理 | 将废塑料破碎、清洗、造粒后重新制成塑料制品，替代原生塑料生产，避免石油开采、精炼和聚合过程的高碳排放。每吨再生PE仅产生0.3~0.6t CO₂e，为原生PE的1/5~1/3。 |
| 技术优势 | 减排效果极佳，技术成熟，产业链完善；再生产品可直接进入现有注塑/吹塑产线；每吨再生塑料可节约石油约2~3吨。 |
| 数据来源 | 中国循环经济协会《废塑料再生利用减污降碳量评估方法》（2024征求意见稿）、北京工业大学国家重点研发计划课题（2024） |

---

### 3.2 化学回收（热解制油）工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-2200~-2800 kg CO₂e/吨**（范围：-1500~-3500kg） |
| 原理 | 通过热解或解聚将混合废塑料转化为燃油或化工单体，重新用于生产塑料或其他化工产品，实现全循环利用。与焚烧发电相比可减少21.5%~50%的碳排放；与原油加工路线相比可减少约40%的碳排放。 |
| 技术优势 | 可处理混合/受污染塑料（物理回收无法处理的）；实现全循环利用（塑料→油→塑料）；碳排放较焚烧降低可达50%。 |
| 数据来源 | 中国循环经济协会《废塑料化学回收技术和产业发展前沿》（2025）、《中国塑料》期刊（2025） |

---

### 3.3 焚烧发电工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-70~-150 kg CO₂e/吨** |
| 原理 | 焚烧产生的热量用于发电，替代化石能源；但塑料中的碳为化石成因，焚烧会直接排放CO₂（每吨约排放2.59~3.03t CO₂），且烟气处理能耗较高，整体减排效果有限。 |
| 技术优势 | 处理简单快速，利用现有焚烧设施。局限性：化石碳直接排放，减排效果为所有塑料处理路径中最差。 |
| 数据来源 | 中国循环经济协会（2025）、《环境科学》期刊"废弃物焚烧处理温室气体排放情景模拟与预测"（2022） |

---


### 3.4 3D打印线材再生工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-1600~-2000 kg CO₂e/吨** |
| 原理 | PET废瓶经清洗、破碎、热熔挤出为1.75mm标准3D打印线材，替代原生PLA/ABS线材。回收线材碳足迹约为原生线材的1/5~1/3。萨塞克斯大学"Precious Bottles"项目实现校园内PET瓶→线材→打印件的本地闭环，契合该校2035净零排放战略。 |
| 技术优势 | 原料免费、设备开源（Polyformer等设计可3D打印自制）；本地回收本地使用，消除运输碳排放；打印品可再次回收，趋近零废弃循环。局限：不同品牌PET瓶材质差异影响线材一致性，层间附着力需参数调优。 |
| 数据来源 | Al Rashid & Koç, Composites Part C: Open Access, "Additive manufacturing for sustainability, circularity and zero-waste: 3DP products from waste plastic bottles"（2024）、萨塞克斯大学"Precious Bottles"试点项目（2023） |

---

### 3.5 生态砖（Ecobrick）工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-1200~-1600 kg CO₂e/吨**（物理封存，碳直接被锁定） |
| 原理 | 将清洁干燥的废软塑料紧密填充于PET瓶中（密度≥0.33 g/ml），制成可承重的建筑模块。塑料中的化石碳被物理封存于瓶内而非焚烧释放，成品用于校园景观长椅、花坛围挡、展示墙等。全球生态砖联盟（GEA）提供免费开源标准与认证体系。 |
| 技术优势 | 零设备投入，纯手工制作，人人可参与；碳封存直观可见（每个生态砖≈封存200~500g塑料）；成品直接用于校园建设，兼具环保教育与实用价值。局限：密度控制需培训，大型承重结构需专业设计配合。 |
| 数据来源 | 全球生态砖联盟 ecobricks.org、马来西亚科廷大学500瓶生态砖校园项目（2023）、湖南彭桥中心完小"废塑料变身生态砖"课题实践（2024） |


### 3.6 塑木复合材料（WPC）工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-1600~-2000 kg CO₂e/吨** |
| 原理 | 将废塑料（HDPE/PP）与木纤维/木粉（40~45%木+50~60%塑）经挤出或热压成型为塑木型材，替代实木或纯塑料建材。全生命周期分析表明WPC托盘碳足迹仅约1,236 kg CO₂-eq/千次运输，为纯HDPE托盘的57%，且寿命达66次循环（木质仅20次）。亚利桑那州立大学CarbonEX项目进一步开发了碳负型WPC，以生物基碳替代石油基塑料。 |
| 技术优势 | 耐久性远超实木（耐腐、不裂、免漆），寿命可达30年；可100%用废弃物生产，单位碳排放为混凝土的27%；产品可再次回收造粒形成闭环。局限：生产需挤出/热压设备，校园级需依托创客空间或校外合作。 |
| 数据来源 | Brunnhuber et al., Springer 2023, "Ex-Ante LCA of WPC Decking" (DOI:10.1007/978-3-031-29294-1_3)、Khan & Horttanainen, LUT University 2023 "WPC Pallets from CDW"、CarbonEX项目（Arizona State University, 2023） |

## 四、有害垃圾处理工艺

### 4.1 资源化回收（金属回收）工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-1500~-11000 kg CO₂e/吨**（视物料：废电路板约-1500~-2000，三元锂电池约-2760~-3750，废铝约-11000） |
| 原理 | 从有害垃圾中回收铜、铅、锌、锂、金等金属，替代原生金属开采和冶炼过程的高能耗、高排放，同时避免重金属污染环境。 |
| 技术优势 | 减排潜力极大（尤其废铝可达-11000kg/t）；替代原生采矿减少生态破坏；回收贵金属具有显著经济价值。 |
| 数据来源 | 中国信息通信研究院（2026）、《环境科学》期刊"废旧三元锂离子电池回收利用碳足迹"（2024） |

### 4.2 水泥窑协同处置工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-700~-800 kg CO₂e/吨** |
| 原理 | 利用水泥窑1400℃以上高温彻底分解有毒有害物质；有害垃圾中的可燃成分替代部分煤炭作为燃料，无机成分替代部分水泥原料，实现"减量化、无害化、资源化"协同。每替代1吨煤炭可减少约0.83吨CO₂排放。 |
| 技术优势 | 高温彻底分解有害物（无二次污染）；可燃成分替代煤炭，无机成分替代水泥原料；利用现有水泥生产线，不需独立建厂。 |
| 数据来源 | 生态环境部环境发展中心（2026）、中国水泥协会（2024） |

### 4.3 高温焚烧工艺

### 4.4 电池分类拆解回收工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-2000~-3800 kg CO₂e/吨**（三元锂电池约-2760~-3750，废铅酸约-1800~-2200） |
| 原理 | 对校园收集的废旧电池按类型（锂电/镍氢/铅酸/碱性）分类后物理拆解，回收锂、钴、镍、锰等有价金属替代原生采矿冶炼。西安交大"锐锋智锂"团队研发带电拆解技术，实现三废零排放；山东科大团队实现电极剥离效率99.8%、浮选效率99%以上。 |
| 技术优势 | 减排潜力大——锂电池回收碳足迹仅为原生采矿的1/3~1/2；贵金属经济价值高（每吨三元锂电池含钴约50~100kg）；校园实验室与宿舍电池集中回收，原料稳定。局限：拆解需专业培训与防护，短路与电解液泄漏有安全风险。 |
| 数据来源 | 澳门科技大学校园电池回收碳减排数据（年回收0.127吨、减排约38 kg CO₂e，2020-2024持续）、西安交通大学"锐锋智锂"团队（中国国际大学生创新大赛2023全国金奖）、山东科技大学"锂想生活"科创团队（《中国科学报》2024年10月报道） |

### 4.5 荧光灯管汞回收蒸馏工艺

| 属性 | 内容 |
|------|------|
| 碳减排量 | **-600~-900 kg CO₂e/吨** |
| 原理 | 废旧荧光灯管在密闭系统中破碎后，通过真空蒸馏（600°C/30min）将荧光粉中的汞气化并冷凝回收。LI等（2023）证实真空条件下氧化汞转化为元素汞更彻底，汞脱除至检测不出水平；蒸馏后荧光粉矿相不变，可同步回收稀土。江西格林循环产业股份有限公司示范项目年处理≥1100万支，汞回收率99%，烟气汞排放≤0.005mg/m³。 |
| 技术优势 | 真空蒸馏比常压低约100°C，节能显著；汞回收率99%+，玻璃与稀土同步回收无二次污染；单支灯管含汞150~3500μg，校园每学期更换数百支灯管即构成可观回收量。局限：破碎过程汞释放峰值可达4500μg/m³，需在负压密闭手套箱内操作。 |
| 数据来源 | LI Tong et al., Energy Reports 2023 Vol.9 Suppl.6 pp.64-72 "Identification of mercury species in spent fluorescent lamps and mercury recovery by distillation"（中南大学/湖南大学）、江西格林循环示范项目工程数据 |
