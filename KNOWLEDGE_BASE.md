## 一、厌氧消化工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-90.82 kg CO₂e / 吨** |
| 原理 | 避免填埋产生的甲烷排放（甲烷温室效应是CO₂的28倍），同时沼气发电替代化石能源，有机肥替代化肥。 |
| 流程 | 厨余垃圾 → 预处理（破碎/除杂） → 厌氧发酵罐（产沼气） → 沼气发电/提纯 → 沼渣制成有机肥 |
| 产物 | 沼气（可发电/提纯天然气）、有机肥 |

### 适用场景
- 大规模集中处理厂
- 厨余纯净度要求较高（杂质应 < 10%）
- 适合与市政燃气管网配套

### 优缺点
- ✅ 减排效果最优，产物附加值高
- ❌ 投资大，建设周期长，对前端分类要求高

## 二、发酵产酸工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-81.04 kg CO₂e / 吨** |
| 原理 | 通过发酵将有机质转化为生物碳源，替代传统碳源生产，减少碳排放。 |
| 流程 | 厨余垃圾 → 水解酸化 → 产酸发酵 → 分离提纯 → 生物碳源产品 |
| 产物 | 生物碳源（替代工业碳源用于污水处理等） |

### 适用场景
- 邻近污水处理厂的项目
- 对沼气利用条件有限的场景

### 优缺点
- ✅ 产物可与污水处理协同，节省工业碳源成本
- ❌ 市场规模受限于周边污水处理需求

## 三、三相分离协同焚烧工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-80.96 kg CO₂e / 吨** |
| 原理 | 分离油脂制生物柴油，固渣焚烧发电，减少填埋和化石能源使用。 |
| 流程 | 厨余垃圾 → 三相分离（油/水/固） → 油脂→生物柴油 \| 固渣→焚烧发电 \| 废水→厌氧处理 |
| 产物 | 生物柴油、电力、再生水 |

### 适用场景
- 与垃圾焚烧厂邻近的项目
- 厨余含油率较高的场景（中餐烹饪油脂多）

### 优缺点
- ✅ 利用已有焚烧设施，降低独立建厂成本
- ❌ 焚烧固渣仍有碳排放，资源化率不及厌氧消化

## 四、黑水虻生物转化工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-41.78 kg CO₂e / 吨** |
| 原理 | 黑水虻幼虫取食厨余垃圾，转化为高蛋白虫体和虫粪有机肥，减少废弃物处理碳排放。 |
| 流程 | 厨余垃圾 → 粉碎预处理 → 黑水虻幼虫养殖（7-10天） → 分离 → 虫体蛋白饲料 + 虫粪有机肥 |
| 产物 | 昆虫蛋白饲料、有机肥 |

### 适用场景
- 小型分散式处理
- 校园/社区级别的就地处理
- 有水产养殖/家禽养殖配套的场景

### 优缺点
- ✅ 投资小、占地少、操作简单，适合校园场景
- ❌ 减排量不及工业级处理，对温度/湿度有要求

## 五、直接混合焚烧工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-3.01 kg CO₂e / 吨** |
| 原理 | 焚烧发电替代部分化石能源，但厨余含水率高导致热值低，减排效果较弱，未充分利用资源化潜力。 |
| 流程 | 厨余垃圾 → 混合其他垃圾 → 焚烧炉 → 余热发电 → 炉渣填埋 |
| 产物 | 电力（热值较低） |

### 适用场景
- 缺乏独立厨余处理设施的地区
- 作为过渡方案，不推荐作为长期策略

### 优缺点
- ✅ 处理简单，利用现有焚烧厂
- ❌ 热值低、含水率高导致焚烧效率差，资源化程度最低

## 六、如何新增工艺

### 1. 编辑本文件

在下面新增一个二级标题，复制上面的表格模板填写即可。

### 2. 更新代码 `src/config.py`

在 `TREATMENT_KNOWLEDGE` 字典中新增一条：

```python
"新工艺Key": {
    "title": "新工艺名称",
    "co2_per_ton": -XX.XX,
    "unit": "kg CO₂e / 吨",
    "principle": "减排原理描述",
    "process": "工艺流程步骤",
    "output": "产物列表",
},
```

### 3. 更新 `src/seed.py`

在 `SEED_FACTORS` 中添加对应的碳因子：

```python
{"category": "外卖厨余", "path": "新工艺Key", "factor": XX.XX, "source": "数据来源"},
```

### 4. 前端无需改动

认领去向 `<select>` 和知识弹窗均从配置动态生成。
