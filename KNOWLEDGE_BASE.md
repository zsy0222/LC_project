## 一、厌氧消化工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-90.82 kg CO₂e / 吨** |
| 原理 | 避免填埋产生的甲烷排放（甲烷温室效应是CO₂的28倍），同时沼气发电替代化石能源，有机肥替代化肥。 |
| 流程 | 厨余垃圾 → 预处理（破碎/除杂） → 厌氧发酵罐（产沼气） → 沼气发电/提纯 → 沼渣制成有机肥 |
| 产物 | 沼气（可发电/提纯天然气）、有机肥 |

### 适用场景
- 大规模集中处理厂
- 厨余纯净度要求较高（杂质应 < 10%）
- 适合与市政燃气管网配套

### 优缺点
- ✅ 减排效果最优，产物附加值高
- ❌ 投资大，建设周期长，对前端分类要求高

## 二、发酵产酸工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-81.04 kg CO₂e / 吨** |
| 原理 | 通过发酵将有机质转化为生物碳源，替代传统碳源生产，减少碳排放。 |
| 流程 | 厨余垃圾 → 水解酸化 → 产酸发酵 → 分离提纯 → 生物碳源产品 |
| 产物 | 生物碳源（替代工业碳源用于污水处理等） |

### 适用场景
- 邻近污水处理厂的项目
- 对沼气利用条件有限的场景

### 优缺点
- ✅ 产物可与污水处理协同，节省工业碳源成本
- ❌ 市场规模受限于周边污水处理需求

## 三、三相分离协同焚烧工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-80.96 kg CO₂e / 吨** |
| 原理 | 分离油脂制生物柴油，固渣焚烧发电，减少填埋和化石能源使用。 |
| 流程 | 厨余垃圾 → 三相分离（油/水/固） → 油脂→生物柴油 \| 固渣→焚烧发电 \| 废水→厌氧处理 |
| 产物 | 生物柴油、电力、再生水 |

### 适用场景
- 与垃圾焚烧厂邻近的项目
- 厨余含油率较高的场景（中餐烹饪油脂多）

### 优缺点
- ✅ 利用已有焚烧设施，降低独立建厂成本
- ❌ 焚烧固渣仍有碳排放，资源化率不及厌氧消化

## 四、黑水虻生物转化工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-41.78 kg CO₂e / 吨** |
| 原理 | 黑水虻幼虫取食厨余垃圾，转化为高蛋白虫体和虫粪有机肥，减少废弃物处理碳排放。 |
| 流程 | 厨余垃圾 → 粉碎预处理 → 黑水虻幼虫养殖（7-10天） → 分离 → 虫体蛋白饲料 + 虫粪有机肥 |
| 产物 | 昆虫蛋白饲料、有机肥 |

### 适用场景
- 小型分散式处理
- 校园/社区级别的就地处理
- 有水产养殖/家禽养殖配套的场景

### 优缺点
- ✅ 投资小、占地少、操作简单，适合校园场景
- ❌ 减排量不及工业级处理，对温度/湿度有要求

## 五、直接混合焚烧工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-3.01 kg CO₂e / 吨** |
| 原理 | 焚烧发电替代部分化石能源，但厨余含水率高导致热值低，减排效果较弱，未充分利用资源化潜力。 |
| 流程 | 厨余垃圾 → 混合其他垃圾 → 焚烧炉 → 余热发电 → 炉渣填埋 |
| 产物 | 电力（热值较低） |

### 适用场景
- 缺乏独立厨余处理设施的地区
- 作为过渡方案，不推荐作为长期策略

### 优缺点
- ✅ 处理简单，利用现有焚烧厂
- ❌ 热值低、含水率高导致焚烧效率差，资源化程度最低

## 六、如何新增工艺

### 1. 编辑本文件

在下面新增一个二级标题，复制上面的表格模板填写即可。

### 2. 更新代码 `src/config.py`

在 `TREATMENT_KNOWLEDGE` 字典中新增一条：

```python
"新工艺Key": {
    "title": "新工艺名称",
    "co2_per_ton": -XX.XX,
    "unit": "kg CO₂e / 吨",
    "principle": "减排原理描述",
    "process": "工艺流程步骤",
    "output": "产物列表",
},
```

### 3. 更新 `src/seed.py`

在 `SEED_FACTORS` 中添加对应的碳因子：

```python
{"category": "外卖厨余", "path": "新工艺Key", "factor": XX.XX, "source": "数据来源"},
```

### 4. 前端无需改动

认领去向 `<select>` 和知识弹窗均从配置动态生成。

## 八、快递纸箱（废纸）处理工艺

### 8.1 回收制浆再生工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-1200 ~ -1600 kg CO₂e / 吨**（再生瓦楞纸约 -1580，再生文化纸约 -1410，再生白板纸约 -950） |
| 原理 | 替代原生木浆生产，减少 4~5m³ 木材消耗及制浆过程 60%~70% 的能源消耗和化学品使用，同时避免填埋产生的甲烷排放。 |
| 流程 | 废纸 → 分拣/去杂 → 碎解 → 脱墨 → 筛选净化 → 抄纸 → 再生纸产品 |
| 产物 | 再生瓦楞纸、再生文化纸、再生白板纸 |

### 8.2 热解制生物炭工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-427 kg CO₂e / 吨** |
| 原理 | 在缺氧环境下将废纸热解为稳定的生物炭，实现长达百年的碳封存；生物炭还可改良土壤，提升土壤碳汇能力。 |
| 流程 | 废纸 → 干燥粉碎 → 高温热解（缺氧） → 生物炭 + 热解气 |
| 产物 | 生物炭（土壤改良剂）、可燃气 |

### 8.3 焚烧发电工艺（废纸）

| 属性 | 内容 |
|------|------|
| 减排量 | **-690 kg CO₂e / 吨** |
| 原理 | 焚烧产生的热量用于发电，替代化石能源；废纸中的碳为生物成因，焚烧排放的 CO₂ 属于碳循环的一部分，不计入化石碳排放。 |
| 流程 | 废纸 → 混合其他可燃垃圾 → 焚烧炉 → 余热锅炉 → 汽轮机发电 |
| 产物 | 电力 |

## 九、塑料处理工艺

### 9.1 物理回收再生（造粒）工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-1900 ~ -2400 kg CO₂e / 吨** |
| 原理 | 将废塑料破碎、清洗、造粒后重新制成塑料制品，替代原生塑料生产，避免石油开采、精炼和聚合过程的高碳排放。每吨再生 PE 仅产生 0.3~0.6t CO₂e，为原生 PE 的 1/5~1/3。 |
| 流程 | 废塑料 → 分拣（按材质） → 破碎 → 清洗 → 熔融造粒 → 再生塑料粒子 → 注塑/吹塑 → 再生产品 |
| 产物 | 再生塑料粒子、再生塑料制品 |

### 9.2 化学回收（热解制油）工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-2200 ~ -2800 kg CO₂e / 吨** |
| 原理 | 通过热解或解聚将混合废塑料转化为燃油或化工单体，重新用于生产塑料或其他化工产品，实现全循环利用。与焚烧相比可减少 21.5%~50% 碳排放。 |
| 流程 | 废塑料 → 预处理 → 热解反应器 → 冷凝分离 → 热解油/化工单体 |
| 产物 | 热解油（可再制塑料）、化工原料 |

### 9.3 焚烧发电工艺（塑料）

| 属性 | 内容 |
|------|------|
| 减排量 | **-70 ~ -150 kg CO₂e / 吨** |
| 原理 | 焚烧产生的热量用于发电，替代化石能源；但塑料中的碳为化石成因，焚烧会直接排放 CO₂（每吨约排放 2.59~3.03t CO₂），整体减排效果有限。 |
| 流程 | 废塑料 → 混合其他垃圾 → 焚烧炉 → 余热发电 |
| 产物 | 电力 |

## 十、有害垃圾处理工艺

### 10.1 资源化回收（金属回收）工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-1500 ~ -11000 kg CO₂e / 吨**（视具体物料：废电路板 ~ -1500~-2000，三元锂电池 ~ -2760~-3750，废铝 ~ -11000） |
| 原理 | 从有害垃圾中回收铜、铅、锌、锂、金等金属，替代原生金属开采和冶炼过程的高能耗、高排放，同时避免重金属污染环境。 |
| 流程 | 有害垃圾 → 分类拆解 → 破碎分选 → 湿法/火法冶金 → 金属回收 |
| 产物 | 再生铜、铝、锂、金等金属 |

### 10.2 水泥窑协同处置工艺

| 属性 | 内容 |
|------|------|
| 减排量 | **-700 ~ -800 kg CO₂e / 吨** |
| 原理 | 利用水泥窑 1400℃ 以上高温彻底分解有毒有害物质；可燃成分替代部分煤炭，无机成分替代水泥原料。每替代 1 吨煤炭可减少约 0.83 吨 CO₂ 排放。 |
| 流程 | 有害垃圾 → 预处理 → 投料 → 水泥窑（1400℃+） → 熟料 → 水泥 |
| 产物 | 水泥熟料 |

### 10.3 高温焚烧工艺（有害）

| 属性 | 内容 |
|------|------|
| 减排量 | **+500 ~ +600 kg CO₂e / 吨**（净增排） |
| 原理 | 通过高温焚烧彻底分解有机污染物，但焚烧过程能耗极高，且会产生飞灰等危险废物需要进一步固化填埋，整体碳排放高于安全填埋。 |
| 流程 | 有害垃圾 → 高温焚烧炉 → 烟气处理 → 飞灰固化 → 安全填埋 |
| 产物 | 飞灰固化体、余热 |

> 注：以上数据为文献综合参考值，实际项目需根据具体规模、设备、运行条件进行 LCA 核算。后续可由环保专业成员补充来源文献。
