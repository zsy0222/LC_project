# 校园低碳回收追踪系统 · 设计参数表

> 所有可调参数集中管理，修改此处即可全局生效。
> 代码配置入口：`src/config.py`

---

## 一、定位校验

| 参数 | 值 | 说明 |
|------|-----|------|
| 最大允许距离 | `200` 米 | 用户与回收点直线距离阈值，超过则拒绝投递 |
| 距离计算公式 | Haversine | 基于地球半径 6371km 的球面距离公式 |
| 定位超时 | `8000` ms | 浏览器获取 GPS 的超时时间 |
| 定位缓存 | `30000` ms | 浏览器 GPS 缓存有效期 |
| GPS 降级策略 | 放行 | 浏览器不支持定位或获取失败时，弹警告但不阻塞 |

---

## 二、反作弊 · 提交冷却

| 参数 | 值 | 说明 |
|------|-----|------|
| 冷却时间 | `30` 秒 | 同一用户在同一回收点的最小提交间隔 |
| 冷却范围 | 同点位 | 按 `qr_code` 匹配（不同点位独立计时） |
| 超限响应 | HTTP 429 | 返回剩余秒数 |

---

## 三、反作弊 · 图片相似度去重

| 参数 | 值 | 说明 |
|------|-----|------|
| 哈希算法 | aHash（平均哈希） | 感知哈希，对缩放/亮度变化鲁棒 |
| 哈希尺寸 | `32×32` 像素 | 灰度图，生成 1024-bit 哈希 |
| 相似度阈值 | `0.15`（15%） | 汉明距离比例 ≤ 15% 判定为同一物品 |
| 比对范围 | 用户最近 `10` 条 | 仅与本人近期提交比对 |
| 比对方式 | 汉明距离 | 1024 bit 逐位异或 |

---

## 四、多物品计数

| 参数 | 值 | 说明 |
|------|-----|------|
| 最小数量 | `1` | 默认值 |
| 最大数量 | `20` | 前端输入上限 |
| 碳减排计算 | `单件减碳量 × 数量` | 多件叠加 |

## 五、AI 识别

| 参数 | 值 | 说明 |
|------|-----|------|
| 模型 | MobileNetV3 Small | torchvision 预训练（ImageNet 权重） |
| 输入尺寸 | 224×224 | 中心裁剪后尺寸 |
| 品类 | 纸箱 / 塑料 / 玻璃 | 由 ImageNet 类别索引聚合 |
| 完整度等级 | 完好 / 轻损 / 破损 / 受潮 | 图像统计启发式规则 |
| 置信度阈值 | `0.6`（60%） | 低于此值提示重拍 |
| 未知拒绝阈值 | 聚合概率 < `0.02` | 三类总分过低 → 返回"无法识别" |
| 品类评分权重 | 品类 60% + 完整度 40% | 综合置信度公式 |
| 降级模式 | mock-heuristic | torch 不可用时自动切换 |

### 完整度评估启发式参数

| 等级 | 判定条件 | 置信度 |
|------|----------|--------|
| 完好 | 边缘密度 ≤ 22 | 0.88 |
| 轻损 | 边缘密度 22~35 | 0.80 |
| 破损 | 边缘密度 > 35 | 0.78 |
| 受潮 | 亮度 < 70 且 标准差 < 35 | 0.75 |

## 七、碳减排计算

### 公式

```
单次减碳量 = carbon_factors[品类][路径] × item_count

其中:
  - 路径 A/B → path = "Reuse"
  - 路径 C   → path = "Recycle"
```

### 碳因子表（kg CO₂e / 件）

| 品类 | 路径 | 系数 | 数据来源 |
|------|------|------|----------|
| 纸箱 | Reuse | `0.35` | 中国造纸协会 / IPCC AR6 |
| 纸箱 | Recycle | `0.15` | 中国造纸协会 |
| 塑料 | Reuse | `0.10` | PAS 2050 简化 |
| 塑料 | Recycle | `0.06` | PAS 2050 简化 |
| 玻璃 | Reuse | `0.30` | PAS 2050 简化 |
| 玻璃 | Recycle | `0.12` | PAS 2050 简化 |

### 兜底默认值（碳因子表查询失败时）

| 品类 | 默认值 |
|------|--------|
| 纸箱 | `0.30` |
| 塑料 | `0.08` |
| 玻璃 | `0.25` |

### AI 预估公式（仅用于前端展示，非最终值）

```
预估减碳 = base[品类] × (1.0 if 路径 A/B else 0.43)

base = {纸箱: 0.35, 塑料: 0.10, 玻璃: 0.30}
```

## 八、批次归集规则

| 参数 | 值 | 说明 |
|------|-----|------|

## 修改指南

1. **改参数**：找到对应表格，修改 `src/config.py` 中同名常量
2. **改碳因子**：修改 `src/seed.py` 中的 `SEED_FACTORS`，删除 `lc_project.db` 重新初始化
3. **改公式**：修改 `src/services/` 下对应服务文件
4. **改阈值**：优先在 `config.py` 修改，本文档同步更新
